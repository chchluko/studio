# **App Name**: VotaCompa

## Core Features:

- Authentication: Verify if user can vote.
- List Voters: Shows available work partners to vote for.
- Voting: Allows user to cast vote for a work partner, votes saved on database.
- Reason Submission: Accepts text input detailing the reasons why the voter is voting for their selected coworker.
- Vote Limit: Enforce a limit to a single vote per user. Verify using user authentication or a session

## Style Guidelines:

- Primary color: A muted blue (#6699CC) to convey trust and professionalism, inspired by the concept of democratic processes.
- Background color: A very light grey (#F0F0F0), nearly white, providing a clean and neutral backdrop that doesn't distract from the voting options.
- Accent color: A soft green (#8FBC8F) to highlight selected options and confirm actions, contrasting with the blue for a positive affirmation.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, offering a clean, modern, and highly readable experience suitable for professional contexts.
- Use simple, recognizable icons to represent each voting option or action, aiding quick navigation.
- Design a straightforward layout that prioritizes usability, with clear, large buttons for actions. Minimize visual clutter to facilitate decision-making.
- Incorporate subtle animations on vote confirmation or when toggling between options to provide gentle feedback.
